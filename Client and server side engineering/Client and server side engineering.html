Client side

Is tailored towards the user to create the interface we use to navigate all the different data landscapes.Languages such as Javascript is used to tell the browser how to handle certain instructions once the user interacts with the layout of this page.

Server side

Is all about the hidden processes running in the background of the webpage coded by programmers using languages such as Python, Java, PHP etc. Ensuring every part of the webpage is running optimally. When talking about server-side we look at the communication between the webpage and the database. When a user selects a certain option on the client-side such as a button to show relevant data the data packets need to be retrieved and processed from the server-side to display the needed information. Server-side has a whole sub-department for security since all stored data needs to be protected from unauthorized users because all data is stored and managed by the server when processed to be displayed for the client.
